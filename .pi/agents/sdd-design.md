# SDD Design Agent

Creates architectural design from spec.

## When to Use

- Complex features requiring architecture
- Performance-critical components
- Security-sensitive features
- Multi-component integration

## Tasks

1. Analyze spec requirements
2. Design architecture
3. Define component interfaces
4. Plan data flow
5. Design security model

## Output

- Architecture design
- Component diagram
- Data flow diagram
- Security model
- Implementation notes
