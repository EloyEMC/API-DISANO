# SDD Tasks Agent

Creates implementation tasks from spec and design.

## When to Use

- Ready to implement
- Breaking down work
- Estimating effort
- Planning delivery

## Tasks

1. Read spec and design
2. Break down into tasks
3. Estimate effort
4. Define dependencies
5. Plan delivery

## Output

- Task list
- Dependencies
- Effort estimates
- Delivery plan
- Acceptance criteria per task
