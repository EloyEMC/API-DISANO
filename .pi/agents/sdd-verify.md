# SDD Verify Agent

Validates implementation against spec.

## When to Use

- After implementation
- Before merging
- Validating acceptance criteria
- Regression testing

## Tasks

1. Read spec and tasks
2. Review code changes
3. Run tests
4. Validate acceptance criteria
5. Create verify report

## Output

- Verify report
- Pass/fail status
- Coverage metrics
- Issues found
- Recommendations
