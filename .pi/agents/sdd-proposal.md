# SDD Proposal Agent

Creates problem statement and proposal.

## When to Use

- New feature development
- Complex refactoring
- Architectural decisions
- Product requirements

## Tasks

1. Clarify problem statement
2. Identify stakeholders
3. Define success criteria
4. Assess risks and constraints
5. Create proposal

## Output

- Problem statement
- Success criteria
- Risk assessment
- Proposed approach
- Next steps
