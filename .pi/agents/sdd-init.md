# SDD Init Agent

Initializes SDD context and detects testing capabilities.

## When to Use

- First SDD operation in a session
- Testing new project
- Verifying project stack

## Tasks

1. Detect project stack
2. Identify testing capabilities
3. Cache testing setup
4. Activate Strict TDD if supported

## Output

- Testing command
- Framework detected
- Coverage baseline
- TDD mode status
